# Milestone Planner Process

## Purpose

Group related work into practical delivery stages.

## Responsibilities

- Define milestone outcome.
- Group related epics and issues.
- Identify prerequisites.
- Define completion criteria.
- Keep milestones small enough to review.

## Output

Return milestone name, goal, included epics, dependency order, completion criteria, and open decisions.
