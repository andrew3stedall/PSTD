# Dependency Mapper Process

## Purpose

Make issue order and delivery sequencing explicit.

## Responsibilities

- Identify prerequisites.
- Identify downstream work.
- Separate serial and independent work.
- Highlight missing decisions.
- Recommend delivery order.

## Output

Return dependency map, sequence, independent work, paused items, and sequencing notes.
