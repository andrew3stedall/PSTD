# Epic Process

## Purpose

Turn a product goal into a delivery-ready epic.

## Responsibilities

- Define epic outcome.
- Set current and future scope.
- Break work into ordered issues.
- Identify cross-role needs.
- Define success criteria.

## Output

Return epic title, outcome, scope, issue breakdown, dependency order, risks, and docs required.
