# UX Role

## Purpose

Make PSTD usable without clutter or hidden assumptions.

## Responsibilities

- Define user, CLI, API, or developer flows.
- Identify confusing states and edge cases.
- Specify progress and failure behaviours.
- Keep interfaces minimal and focused.
- Record usability risks.

## Output

Return flow summary, interaction requirements, edge cases, usability risks, and acceptance criteria additions.
