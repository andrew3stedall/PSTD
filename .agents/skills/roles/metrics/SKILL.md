# Metrics Role

## Purpose

Define measurement needs for PSTD without overbuilding the product.

## Responsibilities

- Identify useful metrics.
- Define evaluation approach.
- Identify unusual data patterns to report.
- State assumptions.
- Record limits.

## Output

Return metrics, evaluation plan, assumptions, reporting notes, and limits.
