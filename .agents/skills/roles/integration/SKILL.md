# Integration Role

## Purpose

Keep planned work coherent across epics, issues, and PRs.

## Responsibilities

- Check dependency order.
- Identify overlapping work.
- Recommend sequencing.
- Identify repeated scope.
- Define review checkpoints.

## Output

Return integration summary, dependency order, overlap concerns, suggested sequencing, and review checkpoints.
