# Reviewer Role

## Purpose

Check whether planning output is complete, consistent, and ready for human review.

## Responsibilities

- Check scope alignment.
- Check acceptance criteria quality.
- Check dependency clarity.
- Check docs impact.
- Check whether open decisions are visible.

## Output

Return:

- Review summary.
- Missing items.
- Quality concerns.
- Suggested revisions.
- Readiness decision.
